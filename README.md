# Amber

A smart eye health app that implements the 20-20-20 rule to protect your eyes during screen time. Built with SwiftUI for macOS and iOS.

## Features

- **20-20-20 Rule** - Automatically reminds you to take 20-second breaks every 20 minutes
- **Strict Mode** - Full-screen overlay ensures you actually take your breaks
- **Calendar Sync** - Automatically skips breaks during calendar events
- **Apple Health** - Logs eye breaks as mindfulness minutes in HealthKit
- **Dynamic Island & Live Activity** - Real-time timer on iOS lock screen and Dynamic Island
- **iCloud Sync** - Settings sync across all your devices
- **Pomodoro-Compatible** - Customizable work/rest durations
- **Menu Bar App** - Lives unobtrusively in your macOS menu bar
- **Home Screen Widgets** - iOS widgets showing timer status

## Tech Stack

- SwiftUI (multiplatform macOS + iOS)
- WidgetKit & ActivityKit
- StoreKit 2
- EventKit
- HealthKit
- CloudKit (NSUbiquitousKeyValueStore)
